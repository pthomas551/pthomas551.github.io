JavaScript Iteration Statements
Name
Info
for
Repeats until given condition evaluates to false. Example:

for (var i = 0; i < 10; i++) {}

do...while
Repeats until condition specified in while evaluates to true. Example:

do {} while (i < 10);

while
Repeats unless condition specified in while evaluates to true. Difference with do/while is that test is performed before, not after, execution of code. Example:

while (i < 10) {}

for...in
Iterates a variable over the properties of an enumerable object. Example:

states = {AL: "Alabama"...WY: "Wyoming"}
for stateAbbreviation in states {}

for...of
Iterates a variable over the values of an enumerable object. Note: newly added to JS, browser support may be inconsistent. Example:

states = {AL: "Alabama"...WY: "Wyoming"}
for stateName in states {}
